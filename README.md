# dati.gov.it (dati-gov-it)

dati.gov.it is a national government open-data portal for Italy running **CKAN**, exposing a standardized catalog API over ~65,388 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/dati-gov-it/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=dati-gov-it-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.10.3

## API
- **dati.gov.it CKAN Action API** — base `https://www.dati.gov.it/opendata/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://www.dati.gov.it/opendata)
